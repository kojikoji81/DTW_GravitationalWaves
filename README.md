# 重力波時系列データ DTWクラスタリングツール

このリポジトリには、LIGO/Virgo/KAGRA (LV-K) の公開重力波時系列データを取得し、DTW (Dynamic Time Warping) ベースのクラスタリング手法 (`tslearn.clustering.KShape`) を用いて、波形パターンを自動的に分類・可視化するためのPythonスクリプトが含まれています。

## 概要

重力波検出器から得られるデータは膨大であり、ノイズ（グリッチ）や潜在的な重力波信号、その他様々な現象が複雑に混在しています。本スクリプトは、DTWの持つ時間軸の歪みを許容する特徴を活かし、似た形状を持つ波形を自動でグループ化することで、データ探索、ノイズ分類、あるいは未知のイベント探索の第一歩を支援します。

デモンストレーション用途を想定しており、データ取得期間やクラスタリング対象のシーケンス数を意図的に限定しています。

## 機能

-   **重力波データ取得**: GWOSC (Gravitational-Wave Open Science Center) から指定された期間の重力波ストレインデータを自動ダウンロード。データ取得失敗時はランダムノイズデータで代替。
-   **データ前処理**: バンドパスフィルターによるノイズ除去と、各時系列シーケンスの標準化。
-   **シーケンス分割**: 連続した時系列データを、オーバーラップなしの固定長ウィンドウで波形シーケンスに分割。
-   **DTWベースのクラスタリング**: `tslearn.clustering.KShape` を用いて、時間軸の歪みを考慮した波形類似度に基づいてデータをクラスタリング。
-   **結果の可視化**: 各クラスタに属する波形と、そのクラスタの代表波形（セントロイド）をサブプロットで表示。クラスタごとに異なる色を使用し、凡例にサンプル数を表示。
-   **画像保存**: 生成されたグラフをPNG画像ファイルとして指定のディレクトリに保存。
